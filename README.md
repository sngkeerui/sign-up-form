## About
This project was largely about applying basic principles in creating and formatting forms using HTML and CSS. I also learnt about the importance of improving client-side experience (e.g. having placeholders and input validation), as well as what happens 'under-the-hood' when a form is submitted.

I also used a very vanilla way of matching the passwords by using an `if...else` statement with JavaScript.
